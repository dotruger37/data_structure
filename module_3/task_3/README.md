## Алгоритм Дейсктры

**Релаксация ребра** - это проверка того, что дает ли продвижение по данному ребру 
новый кратчайший путь к конечной вершине

```C++
bool relax(u, v) {
    if (d[v] > d[u] + w(u, v) {
        d[v] = d[u] + w(u, v);
        pi[v] = u;
        return true;
    }
    return false;
}

```
